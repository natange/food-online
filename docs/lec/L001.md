## Django template setup

---

### Instructions
0. First steps
```sh
uv run python manage.py migrate
```

1. create folder **templates**
2. create file `~/templates/home.html`
3. change on file `core/views.py`:

```python
def home():
    return render(reques, 'home.html')
```

4. change on `core/settings.py`: 
```py
templates :[
    {
        ...
        'DIRS':['name_of_folder','templates'],
        ...	
    }	
]
```

5. How to create superuser 

on terminal seguir las instrucciones y probar.:

```sh
python manage.py createsuperuser
```

With uv
```sh
uv run python manage.py createsuperuser
```
