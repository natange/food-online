
## 3.3 Homepage

1. Copy all code from file `index.html` from template `foodBakery` to `home.html`
2. create a folder `static` on `src/static`
3. copy all folders assets from template `foodBakery` dentro de la 
carpeta `src/static` dentro de la carpeta code
4. config on the file `settings.py` on below line static :
```py
	STATIC_ROOT = BASE_DIR /'static'
	STATICFILES = [
    		'core/static'
	]
```
5. add line at begining from file home.html : 
```html
{% load static %}
```
6. configurar el archivo `settings.py` para habilitar la carpeta PATH
7. luego en la vista que queremos renderizar html anadir
```html
{% load static %}
```

## 3.4 Collect static files
setting apply command: 
```bash
python manage.py collectstatic
```
